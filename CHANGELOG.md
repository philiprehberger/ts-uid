# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- `uid()` function for generating time-sortable base62 unique IDs
- `extractTime()` to decode timestamp from an ID
- `isValid()` for ID format validation
- Configurable length and prefix support
- Monotonic counter for same-millisecond uniqueness
